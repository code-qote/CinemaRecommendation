Мое приложение подбирает фильм по вашему вкусу среди тех, что идут сейчас в кино.

В папке data находятся базы данных и картинки, которые создаются при помощи парсера.

Перед запуском откройте в cmd папку с проектом и выполните команду: python -m pip install -r requirements.txt
Запускать необходимо файл main.py

Фильмы выдаются от наиболее подходящих к наименее подходящим
Если Вы знаете, какой фильм Вам нужен, следует писать в поле "Поиск по названию фильма". Ввод в этом поле автоматически блокирует выбор других фильтров кроме выбора района.
Если по вашим фильтрам ничего не нашлось, программа специально выдаст все фильмы

Попробуйте заполнить данные быстро
